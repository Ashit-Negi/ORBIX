# ORBIX 🚀

ORBIX is a modern full-stack developer community platform where users can connect, share posts, join communities, chat in real-time, and interact with other developers.

Built with Next.js, Node.js, Prisma, PostgreSQL, and Socket.io.

---

## ✨ Features

### 🔐 Authentication

- User Registration & Login
- JWT Authentication
- Protected Routes
- Persistent Sessions

### 📝 Posts System

- Create & Delete Posts
- Global Feed
- Community Posts
- Real-time Updates

### 💬 Comments & Replies

- Add Comments
- Nested Replies
- Like Comments
- Real-time Comment Updates

### 👥 Communities

- Create Communities
- Join Communities
- Community Feed
- Member Counts

### 🤝 Connections

- Send Connection Requests
- Accept / Remove Connections
- Suggested Users System

### 📩 Real-time Messaging

- 1-on-1 Chat
- Online Users
- Instant Message Delivery
- Socket.io Integration

### 🔔 Notifications

- Real-time Notifications
- Notification Dropdown
- Mark as Read

### 🖼️ Profile System

- Dynamic Profiles
- Edit Profile
- Experience Section
- Profile Images
